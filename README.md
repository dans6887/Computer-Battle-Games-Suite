# Computer-Battle-Games-Suite
An updated collection of all the games from the 1982 Computer Battle Games Book

A recreation suite of all the games from Computer Battle Games

In 1982 usborn Computer Programs published Computer Battlegames. Written by Daniel lsaaman and Jenny Tyler, the book was designed to teach programming for microcomputers such as the ZX81, ZX Spectrum,BBC,VIC20,TRS-80 and Pet and Apple micros. The games were very simple and taught aspiring programmers the fundamentals of using the BASIC coding language. A pdf copy of the book can be downloaded for from colorcomputerarchive.com at the followng link. https://colorcomputerarchive.com/repo/Documents/Books/Computer%20Battlegames%20(Usborne%20Publishing).pdf

These games were written in python. They performs the same functions with some variation on message outputs and the use of imported libraries. View the README file in the suite to see how to get started.
